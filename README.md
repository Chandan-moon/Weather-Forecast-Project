# Weather-Forecast-Project

A modern, responsive **Weather Forecast Web Application** built using **HTML, CSS (Tailwind CSS), and JavaScript**, powered by the **OpenWeather API**.

The app provides **real-time weather data**, **5-day forecasts**, **location-based weather**, **dynamic weather images**, and a **glassmorphism user interface**.

---

## Features

- Search weather by city name
- Get weather using current location
- Temperature unit toggle (Celsius / Fahrenheit)
- 5-day weather forecast
- UI using Tailwind CSS
- Dynamic weather images based on weather condition

- Recent search history using LocalStorage
- Clear search history option
- Error handling for invalid city names
- Fully responsive design

---

## Technologies Used

- HTML5
- CSS3
- Tailwind CSS
- JavaScript (ES6)
- OpenWeather API
- Browser Geolocation API
- LocalStorage

---

## Notes

- Add **OpenWeather API key** inside the `script.js` file before running the project if the API is not responding.
- Allow browser location permission to use the current location feature.
- Weather images change dynamically based on:
  - Weather condition
  - Day or night status
- Recent searched cities are stored using **LocalStorage**.
